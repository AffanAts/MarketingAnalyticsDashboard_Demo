# CRM Digital Marketing Analytics Platform

Platform analitik CRM untuk mengukur dan memvisualisasikan efektivitas digital marketing melalui pendekatan marketing funnel 6 tahap.

## Features

- Marketing Funnel visualization (Awareness → Interest → Consideration → Purchase → Retention → Advocacy)
- Inquiry management (CRUD, follow-up tracking, stage advancement, drop/restore)
- Channel performance analytics (Instagram, Website, TikTok, Facebook, WhatsApp)
- Website DA/PA monitoring with SEO-Inquiry correlation chart
- Configurable funnel dropout thresholds
- User management with role-based access
- Audit log tracking all system activities

## Tech Stack

- React 18
- Tailwind CSS v3
- Recharts
- React Router v6

## Run Locally

```bash
npm install
npm start
```

## Demo Mode

This is a demo version with simulated data. All CRUD operations work in-memory only — data resets on page refresh. No backend, no API calls, no login required.

The app starts directly as `super_admin` with full access to all features.
